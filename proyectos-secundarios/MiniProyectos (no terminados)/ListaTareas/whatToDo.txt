Voy a hacer una Lista de Tareas. 
Cada tarea tendrá un título y un texto. Se podrá crear, modificar y eliminar tareas.
Las tareas creadas tendrán fecha y hora de creación. En caso de que una tarea sea modificada, tanto la fecha como la hora se actualizarán.

Por el lado de HTML/CSS, voy a aprovechar para practicar con Flexbox & Grid.

------------------------------------------------

Pasos a seguir:

    1- Crear la estructura HTML.
    2- Modificar con CSS, usando Flexbox & Grid.
    3- Dejarlo simple e intuitivo visualmente.
    4- Arrancar con el código JS para la Lista de Tareas, utilizando arrays, objetos y demás.
    5- Una vez implementado el JS, utilizar Luxon para manipular las fechas y horas.
